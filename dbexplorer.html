Exemplos de Comandos

                            

                                

                                    
Buscar todos os documentos de uma coleção (SELECT *)

                                    
{ "action": "get", "collection": "products" }

                                

                                

                                    
Buscar um documento por ID (SELECT ... WHERE ID=)

                                    
{ "action": "get", "collection": "users", "docId": "uid_do_usuario" }

                                

                                 

                                    
Buscar com filtros (SELECT ... WHERE)

                                    
{ "action": "get", "collection": "professionals", "where": [["status", "==", "inactive"]] }

                                

                                

                                    
Atualizar um documento (UPDATE)

                                    
{ "action": "update", "collection": "services", "docId": "id_do_servico", "data": { "price": 99.90, "active": false } }

                                

                                

                                    
Apagar um documento (DELETE)

                                    
{ "action": "delete", "collection": "appointments", "docId": "id_do_agendamento" }
